Le roi liche

Invoquez un monstre sur votre terrain depuis n'importe quel cimetierre. Cet effet peut être utilisé une fois par tour.

Insensible à tout les effets, cet effet ne peut être enlever. Une fois durant votre tour, vous pouvez bannir une carte magie de votre deck ou cimetierre puis ajoutez pour ce tour son effet à Balthazar l'Hérétique.

5

Sleiphnir, Joyeau de l'Omniscient

Lorsque cette carte est invoqué, vous pouvez ajoutez à votre main une carte "Balthazar" ou le mentionnant. Une fois par tour, vous pouvez bannir une carte de votre main ou terrain, ajoutez une carte de votre cimetierre dans votre main mais banissez la lors de sa prochaine activation lorsqu'elle quitte le terrain.

Artéfact

Rage du Magus Suprème

Détruisez tout les monstres sur le terrain, si "Balthazar" est sur le terrain au moment de l'activation, détruisez aussi toutes les cartes Magies/Artefactes de votre adversaire. Annulez l'effet de cette carte si vous avez attaquez le tour où vous utilisez "Rage du Magus Suprême".

Magie

Balthazar Orcus, Le Conjuré

Lorsque "Balthazar Orcus, Le Conjuré" "est invoqué sur le terrain, vous pouvez ajoutez à votre main une carte "Ascension" ou d'archétype "Héralté". Lorsque vous utilisez une carte "Héralté", vous pouvez utilisez les monstres dans votre cimetierre mais si vous le faites, bannissez les cibles.
